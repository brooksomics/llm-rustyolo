# Release v0.5.1 - Maintenance Release

**Release Date:** January 13, 2026

This is a maintenance release focused on dependency updates, build tooling improvements, and license compliance.

## üîÑ What's Changed

### Dependency Updates

This release includes comprehensive dependency updates across the entire stack:

**Rust Dependencies:**
- Updated `reqwest` from 0.12 to 0.13 for improved HTTP client performance and security
- Updated `self_update` to 0.42 for better auto-update functionality
- Updated `dirs` to 6.0.0 for improved cross-platform directory path handling
- Updated `toml` to 0.9 for configuration file parsing

**GitHub Actions Workflows:**
- Updated `actions/cache` to v5
- Updated `actions/checkout` to v6
- Updated `actions/setup-python` to v6
- Updated `docker/build-push-action` to v6

**Docker Base Image:**
- Updated to `node:25-slim` for latest Node.js LTS and reduced image size

### License Compliance

Fixed cargo-deny license validation errors by expanding the allowed license list:
- Added OpenSSL license (for OpenSSL dependency chain)
- Added ISC license
- Added CDLA-Permissive-2.0 license

This ensures full compliance with our security audit tooling while maintaining our strict open-source license requirements.

## üîí Security

All dependency updates include the latest security patches and improvements. No new vulnerabilities introduced.

## üì¶ Installation

### Homebrew (macOS/Linux)
```bash
brew upgrade rustyolo
docker pull ghcr.io/brooksomics/llm-rustyolo:latest
```

### Manual Installation
```bash
rustyolo update --binary --yes
rustyolo update --image
```

Or build from source:
```bash
git clone https://github.com/brooksomics/llm-rustyolo.git
cd llm-rustyolo
git checkout v0.5.1
cargo build --release
docker build -t ghcr.io/brooksomics/llm-rustyolo:latest .
```
