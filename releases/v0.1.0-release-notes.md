### Installation

#### Quick Start (Linux x86_64)

1. **Download the binary:**
   - Download `rustyolo-x86_64-unknown-linux-gnu.tar.gz` from the assets below
   - Extract: `tar -xzf rustyolo-x86_64-unknown-linux-gnu.tar.gz`
   - Move to PATH: `sudo mv rustyolo /usr/local/bin/`

2. **Build the Docker image:**
   ```bash
   git clone https://github.com/brooksomics/llm-rustyolo
   cd llm-rustyolo
   docker build -t llm-rustyolo:latest .
   ```

3. **Run it:**
   ```bash
   rustyolo --allow-domains "github.com api.github.com" \
            -v ~/.ssh:/home/agent/.ssh:ro \
            --auth-home ~/.config/rustyolo \
            claude
   ```

#### From Source

```bash
git clone https://github.com/brooksomics/llm-rustyolo
cd llm-rustyolo
cargo build --release
sudo cp target/release/rustyolo /usr/local/bin/
docker build -t llm-rustyolo:latest .
```

### Usage Examples

**Basic usage with network access:**
```bash
rustyolo --allow-domains "github.com api.github.com pypi.org" \
         -v ~/.ssh:/home/agent/.ssh:ro \
         -v ~/.gitconfig:/home/agent/.gitconfig:ro \
         --auth-home ~/.config/rustyolo \
         claude
```

**Completely offline (no network):**
```bash
rustyolo -v ~/.ssh:/home/agent/.ssh:ro \
         --auth-home ~/.config/rustyolo \
         claude
```

**Update rustyolo:**
```bash
rustyolo update
```
